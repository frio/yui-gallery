Menu.Item Class

Extends Tree.Node
Defined in: src/sm-menu/js/menu-item.js:10
Module: gallery-sm-menu-item
Parent Module: gallery-sm-menu

Represents a single menu item in a Menu.

Constructor

Menu.Item

(
  • menu
  • [config]
)

Defined in src/sm-menu/js/menu-item.js:10

Parameters:

  • menu Menu

    Menu instance with which this node should be associated.

  • [config] Object optional

    Configuration hash for this menu item. Supports all the config properties of Tree.Node in addition to the following.

    • [state] Object optional

      State hash for this menu item.

      • [disabled=false] Boolean optional
        If true, this menu item will be disabled, and will not be clickable or selectable.
      • [hidden=false] Boolean optional
        If true, this menu item will be hidden.
    • [type='item'] String optional

      Type of this menu item. May be 'item', 'heading', or 'separator'.

    • [url='#'] String optional

      URL associated with this item. If this item is of type 'item', clicking on the item will navigate to this URL.
